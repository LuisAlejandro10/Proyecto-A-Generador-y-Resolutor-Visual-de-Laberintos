# Proyecto-A-Generador-y-Resolutor-Visual-de-Laberintos
El proyecto es la recopilación de todos los temas vistos en el curso de Estructura de datos de la Beneméritas Universidad Autónoma de Puebla. Recopila toda los temas vistos para la generación aleatoria grafica de un laberinto y su resolución por recesividad  y aprender a realizar trabajos en equipo
